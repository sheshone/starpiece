# 地形、神域与前期章节机制

本文件对应当前代码实现。所有可调数值集中在：

```text
scripts/data/game_definitions.gd
```

## 当前主胜利条件

胜利条件已调整为：

```text
摧毁全部敌方核心，且中央核心仍然存活。
```

地图填充率不再是强制通关条件，而是评分项目。评分主要参考：

- 地图完成基础分
- 地图填充率
- 激活锚点数量
- 剩余核心生命
- 剩余神力
- 通关时间
- 污染崩塌数量

这样玩家打爆敌方核心后即可进入结算；继续补满地图属于高分追求，不再拖慢节奏。

## 前七章教学节奏

本游戏不单独制作教程模式。教学直接融入前期关卡：

- 每章只引入一个主要新概念。
- 已学习内容持续复用。
- 少量提示配合关卡结构，让玩家主动发现机制。

### 第一章：失落平原

解锁：

- 地形：平原
- 神祇：疾野神、盎然神
- 敌方核心：2 座

教学目标：

- 购买地块
- 放置地块
- 形成神域
- 建造神祇
- 开始战斗

提示：

```text
将地块放置在核心附近。
相连的同类地块形成神域。
神域中可以建造神祇。
```

### 第二章：山脊要塞

新增：

- 地形：山地
- 神祇：轰爆神、泞滞神

教学目标：

- 山地不可通行
- 利用山地改变敌人路线
- 山地不能封死敌方核心到中央核心的路线

提示：

```text
山地会阻挡敌人，但不能封死路线。
```

### 第三章：奔流峡谷

新增：

- 地形：河流
- 神祇：澜沧神、漩涡神

教学目标：

- 河流冲送
- 利用河流控制敌人位置

提示：

```text
敌人首次进入河流时会被冲向最近岸边。
```

### 第四章：迷雾森林

新增：

- 地形：森林
- 神祇：蛊郁神、丰饶神

教学目标：

- 森林混乱
- 资源生产

提示：

```text
敌人首次进入森林时会短暂迷失方向。
```

### 第五章：神域共鸣

新增：

- 神域邻接机制

教学目标：

```text
不同地形相邻，会改变神祇技能效果。
```

点击神祇时，应能看到当前邻接地形和获得的技能变化。

### 第六章：神域觉醒

新增：

- 大型神域

教学目标：

```text
神域达到 6 格后，解锁战斗中每阶段一次的主动技能。
```

### 第七章：四域战争

解锁：

- 全部地形
- 全部八位神祇
- 全部邻接机制
- 大型神域

此章结束后，视为完成新手阶段。

## 地形规则

- 山地：完全不可通行，永久或临时山地都必须保留敌方核心到中央核心的路线。
- 平原：敌人受到的所有伤害提高。
- 河流：敌人首次进入连通河流时，按照进入方向横向寻找最近合法岸边并冲送。
- 森林：敌人首次进入连通森林时，在森林内进行有限次数随机移动。
- 混沌：正常通行，没有额外效果。

河流岸边允许使用混沌格。岸边只排除地图外、山地、河流本身、玩家核心、敌方核心、神祇格和其他明确不可通行格。

正常寻路不会反弹；山地会被寻路直接绕开。只有河流冲送、漩涡吸引、抛离、森林混乱等外力位移会在撞上山地、核心或地图边界时改向反弹。

## 八位神祇

| 地形 | 征祇 | 佑祇 |
|---|---|---|
| 平原 | 疾野神：高速单体攻击 | 盎然神：治疗最低生命友军 |
| 山地 | 轰爆神：远程范围炮击 | 泞滞神：减速、冻结、易伤 |
| 河流 | 澜沧神：多目标弹射 | 漩涡神：吸引、策反、抛离、沉默 |
| 森林 | 蛊郁神：叠毒与传播 | 丰饶神：资源、利息、免费刷新 |

神域邻接只改变上述技能行为。神祇普通属性只由等级配置提高。

- `score_weights.fill_ratio`：地图填充率评分权重。
- `large_domain_threshold`：大型神域阈值，默认 6 格。
