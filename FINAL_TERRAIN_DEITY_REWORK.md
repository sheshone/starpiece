# 地形、神域与八神最终机制

本文件对应当前代码实现。所有数值集中在：

```text
scripts/data/game_definitions.gd
```

## 已删除的旧规则

- 神域面积不再增加伤害、生命、产量、速度或范围。
- 普通神域邻接不再直接增加攻速、射程、产量、生命或防御。
- `domain_area_multiplier()` 仅保留兼容接口，始终返回 `1.0`。
- 旧共鸣配置键暂时保留以兼容旧数据，但运行时不再读取。

## 地形

- 山地：完全不可通行，永久或临时山地都必须保留敌方核心到中央核心的路线。
- 平原：敌人受到的所有伤害提高。
- 河流：敌人首次进入连通河流时，按照进入方向横向寻找最近合法岸边并冲送。
- 森林：敌人首次进入连通森林时，在森林内进行有限次数随机移动。
- 混沌：正常通行，没有额外效果。

河流岸边允许使用混沌格。岸边只排除地图外、山地、河流本身、玩家核心、敌方核心、神祇格和其他明确不可通行格。

敌人允许在同一格堆叠；锁定和受击优先作用于当前前排敌人，前排死亡或离开后由同格下一名敌人补位。

正常寻路不会反弹；山地会被寻路直接绕开。只有河流冲送、漩涡吸引、抛离、森林混乱等外力位移会在撞上山地、核心或地图边界时改向反弹。

森林混乱默认持续 4 秒、最多随机移动 6 步。每一步都会重新打乱四个方向，不再按照固定顺序走遍森林。

## 八位神祇

| 地形 | 征祇 | 佑祇 |
|---|---|---|
| 平原 | 疾野神：高速单体攻击 | 盎然神：治疗最低生命友军 |
| 山地 | 轰爆神：远程范围炮击 | 泞滞神：减速、冻结、易伤 |
| 河流 | 澜沧神：多目标弹射 | 漩涡神：吸引、策反、抛离、沉默 |
| 森林 | 蛊郁神：叠毒与传播 | 丰饶神：资源、利息、免费刷新 |

神域邻接只改变上述技能行为。神祇普通属性只由等级配置提高。

## 大型神域

默认达到 `large_domain_threshold = 6` 格后解锁，每个战斗阶段一次：

- 平原“一望无垠”：整片平原成为临时攻击或治疗起点。
- 山地“地壳隆起”：周围合法地形临时变为山地，逐格检查路线。
- 河流“洪流”：邻接区域临时变为河流，并淹没或重伤敌人。
- 森林“迷林徘徊”：神域内敌人每一步重新随机方向；撞到山地、核心或边界时反弹，步数耗尽后恢复正常寻路。

点击达到阈值的神祇，可在单位操作框中发动技能。临时地形在持续时间结束或战斗结束后恢复。

## 测试入口

项目冒烟脚本：

```text
res://tools/smoke_current.gd
```

它检查基础场景、商店交互、迁移、移除、战斗结束条件、神域倍率删除、等权寻路和基础平原暴露。

## 手动测试清单

### 地形

1. 用山地尝试封死任一存活敌方核心，确认放置被拒绝。
2. 让敌人进入横向两侧距离不同的河流，确认被冲向较近岸边；距离相同时可随机。
3. 比较同一伤害攻击平原内外敌人，确认平原内承受更多伤害。
4. 让敌人首次进入多格森林，确认只在当前森林内有限随机移动，随后恢复寻路。

### 征祇

- 疾野神：分别邻接山、河、林，观察同目标伤害叠加、攻击间隔缩短与新目标首击。
- 轰爆神：邻接一至三种异类神域，观察爆炸覆盖半径逐步扩大。
- 蛊郁神：分别邻接原、山、河，观察叠毒、毒伤提升与死亡传播。

### 佑祇

- 盎然神：制造多名受伤神祇，确认优先治疗生命比例最低者；再测试护盾、扩散治疗和减伤。
- 泞滞神：分别邻接原、河、林，观察减速叠层冻结、增强减速和易伤状态。
- 澜沧神：分别邻接原、山、林，观察弹射距离、分叉目标和连续次数变化。
- 漩涡神：分别邻接原、山、林，观察策反、向外抛离和沉默。
- 丰饶神：确认只有丰饶神生产资源；邻接平原后结算利息，邻接河流后可能累计免费刷新。

### 大型神域

1. 将任一神域扩张到配置阈值。
2. 进入战斗并点击神祇，确认出现对应主动技能。
3. 同一战斗阶段只能发动一次。
4. 下一战斗阶段恢复使用次数。
5. 山地与河流产生的临时地形必须自动恢复。
