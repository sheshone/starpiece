# 当前系统补丁说明

## 敌方核心

- 敌方核心所在格是永久保留格。无论核心是否已经被摧毁，都不能被地块、山地或作弊填充覆盖。
- 山地仍然不可通行；放置山地时必须保证每个存活敌方核心到中央核心仍有路线。
- 敌方核心只有在其相邻、位于地图内的可用格都已经铺设地形后，才会成为攻击神祇的目标。
- 如果当前已激活的敌方核心全部被提前摧毁，而地图上仍有未激活核心，则立即提前激活下一批核心。

## 选中与攻击范围

- 攻击范围只在直接点击神祇或敌人后显示。
- 选中神祇时只标记神祇所在格，不再给整片神域绘制选择框。
- 开始放置地块时会立即清除单位选中状态和攻击范围。

## 存档

- 游戏只在建设阶段保存检查点，避免保存正在移动或攻击中的临时战斗状态。
- 检查点包含当前地图、轮数、核心生命、神力、祝福、统计、地形、污染、锚点、神祇和敌方核心状态。
- 主菜单的“星球”页面在存在检查点时显示“继续当前星球”。
- 地图完成或游戏失败后清除当前检查点；历史星球与本地记录继续保留。
- 存档文件仍为：

```text
user://progress.json
```

## 资源神祇

- 资源神祇下方新增生产进度条。
- 战斗阶段中，进度从空逐渐填满；完成生产后重新开始。

## 调试

- F4 作弊面板提高到最上层，并阻止鼠标点击穿透到地图或其他界面。
