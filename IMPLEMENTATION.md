# 拼图神祇塔防 v0.4：实现记录

当前项目已从玩家单位战棋回合改为“拼图建设＋自动塔防”。

## 当前核心系统

- 增加独立开始游戏页面，实际游戏状态在点击开始后初始化。
- 常驻操作改为分布在屏幕四周的紧凑浮卡，地图保持居中。
- 地块与神祇信息跟随所选格以浮动卡显示，事件消息使用短时提示。
- 地形贴图按格随机进行四向旋转；地图支持滚轮缩放与 F11 全屏。
- 核心、神祇和敌人的数值改用悬停信息与小型生命条，避免文字覆盖贴图。
- 商店卡牌复用地形贴图，名称与价格居中显示，不再绘制冗余描述。
- 已购买但尚未放置的地块在切换卡牌、点击已有地块、刷新、按 Esc 或开战时退回原商店槽并返还神力。
- 建设阶段不限制地块购买、放置和免费刷新次数。
- 五格商店；买走卡牌后对应槽位留空，刷新时五格全部重新生成。
- 商店可锁定；锁定后下一建设阶段不会自动免费刷新。
- 无限卡池，允许同一商店出现完全相同的卡牌。
- 商店只生成地块牌；购买后立刻进入地图放置，不经过保留区。
- 点击已有空地块时，在该格附近显示两种地形形态神祇及实时邻域数据。
- 神祇按脚下地形自动使用形态名称，并直接消耗神力安置。
- 地块形状使用固定非均匀权重刷新，四种地形仍独立等概率。
- 攻击神祇普通攻击不消耗神力。
- 攻击神祇周围河流会缩短攻击间隔。
- 资源神祇先按基础价格购买，放置时根据周围河流返还神力。
- 污染由每个地图格独立记录，达到上限只摧毁该格及格上神祇。
- 多格拼图损失一个格子后，其余格子和神祇继续保留。
- 建设阶段与固定时长自动战斗阶段。
- 敌人四向 BFS 寻路、绕过山地并攻击阻路神祇或核心。
- 核心归零失败；除核心外地图全部填满胜利。

## 集中配置

- 商店价格、商店卡牌类型权重、格子污染上限和神祇数值：
  `scripts/data/game_definitions.gd`
- 地块形状与固定非均匀概率：
  `scripts/data/piece_shape_config.gd`
- 正式贴图、动画和音效路径：
  `scripts/autoload/asset_catalog.gd`
- 音效播放与缺失资源静默回退：
  `scripts/autoload/audio_manager.gd`
- 完整素材目录、规格和命名清单：
  `ASSET_GUIDE.md`
